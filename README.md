📰 Conecta News

## 📋 Sobre o Projeto

Este é o site da agência de jornalismo [Conecta News], desenvolvido como parte do Projeto Integrador do Curso Técnico em Desenvolvimento de Sistemas.

O site contém notícias sobre o cotidiano escolar, além de conteúdo multimídia como vídeos e podcasts produzidos pelos alunos.

## 🚀 Como acessar

O site está publicado no GitHub Pages e pode ser acessado pelo link:
🔗 [https://Sartinimatheus.github.io/agencia-noticias](https://sartinimatheus.github.io/trabalho-conecta-news/)

## 🛠️ Tecnologias utilizadas

- **HTML5** - Estrutura das páginas
- **CSS3** - Estilização e layout responsivo
- **JavaScript** - Interatividade e funcionalidades
- **GitHub Pages** - Hospedagem gratuita
